# 如何获取你的GitHub Personal Access Token (classic)

> English: [How to get your PAT](githubtoken_en.md)

打开：https://github.com/settings/tokens/new
按下图设置， Note随意命名，Expiration选择No Expiration，以免过期。给repo和user权限。
![alt text](images/token1.png)
![alt text](images/token2.png)

点击Generate token按钮生成
![alt text](images/token3.png)
复制Token到登录页面中的PAT输入框，点击登录。
注意：PAT 正是 Personal Access Token 的缩写，保管好PAT 请不要告诉任何人！
![alt text](images/token4.png)
